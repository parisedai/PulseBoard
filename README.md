# PulseBoard

AI-powered interview intelligence platform that tracks real-time company activity across GitHub, Reddit, news, and hiring trends.

## Features

- Real-time company tracking across engineering and public signals
- AI-generated company summaries and trend analysis
- Semantic search powered by vector embeddings
- Hiring trend aggregation from job postings
- GitHub activity monitoring and repository insights
- Centralized dashboard for interview preparation

## Tech Stack

**Frontend**
- React + Vite

**Backend**
- FastAPI
- PostgreSQL
- Redis
- pgvector

**AI / Data**
- Claude API
- Semantic embeddings
- Retrieval pipelines

**Infrastructure**
- AWS EC2 / RDS / ElastiCache
